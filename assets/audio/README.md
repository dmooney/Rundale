# Audio Assets

Ambient sound files for the Parish audio system.

## Format

- **Codec**: OGG Vorbis (`.ogg`)
- **Sample rate**: 44100 Hz
- **Channels**: Mono or stereo
- **Looping files**: Should crossfade cleanly at loop boundaries
- **Total budget**: <20 MB for all audio assets combined

## Licensing

All audio files must be:

- **CC0** (public domain), or
- **CC-BY** (attribution required — add to `CREDITS.md`)

See `docs/research/ambient-sound-sources.md` for curated source recommendations.

## Directory Structure

| Directory | Content |
|-----------|---------|
| `pub/` | Fiddle reels, sean-nos singing, crowd murmur, glasses, hearth |
| `church/` | Bell tolls (Angelus, Sunday), hymns, stone echo |
| `farm/` | Rooster, cattle, sheep, dogs, donkey, hens |
| `water/` | Water lapping, wind in reeds, waterfowl |
| `bog/` | Wind over bog, curlew calls, bog silence |
| `weather/` | Rain (light/heavy), wind, thunder, fog silence |
| `village/` | Rooster, children playing, doors/footsteps, evening settling |
| `crossroads/` | Dance music, wind |
| `fairy_fort/` | Hawthorn wind, uncanny silence, Samhain atmosphere |
| `nature/` | Generic birds, wind, insects (shared across locations) |
